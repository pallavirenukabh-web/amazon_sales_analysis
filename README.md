Amazon Sales Analysis Project
Project Overview
This project analyzes Amazon ecommerce sales data using:
- MySQL Workbench
- Power BI
- Excel

The goal of this project is to identify:
- Top performing cities
- Highest sales categories
- Profit trends
- Customer sales analysis
- Discount impact on profit



Tools Used
- MySQL Workbench 8.0
- Microsoft Power BI
- Microsoft Excel
- GitHub


Dataset Used
The dataset contains:
- Customer ID
- Customer Name
- Category
- Sales
- Profit
- Quantity
- Discount
- City
- State
- Region

---

SQL Analysis Performed

The following SQL queries were used for analysis:

- Top 5 cities with highest sales
- City with highest profit
- Category with highest profit
- Category with lowest profit
- Region with top sales
- Average sales analysis
- Discount analysis
- Quantity analysis
- Customer sales analysis

Key Insights

- Some cities generated significantly higher sales than others.
- Technology category contributed high sales and profit.
- Higher discounts affected profit in certain regions.
- Customer purchasing behavior varied across categories.
